Evento onChange

Nessa página você ira aprender um pouco sobre o evento onChange

Como funciona o evento onChange ?

O evento onChange é disparado quando o valor de um elemento HTML é alterado, geralmente em um formulário. Ele é comumente usado em elementos de formulário como input, select e textarea. Por exemplo, quando o usuário altera o valor de um input de texto, seleciona uma opção diferente em um select ou altera o texto em um textarea, o evento onChange é acionado. Isso permite que você execute ações com base na alteração do valor do elemento.

Abaixo temnos um exemplo do evento OnChange onde quando o usuario selecionar uma cidade sera mostrado um alerta:

Exemplo

HTML:

<form>
   <label for="cidade">Escolha uma cidade:</label>
   <select id="cidade">
    <option disabled selected>Selecione uma cidade</option>
    <option value="São Paulo">São Paulo </option>
    <option value="Rio de Janeiro">Rio de Janeiro </option>
    <option value="Brasília">Brasília </option>
    <option value="Salvador">Salvador </option>
    <option value="Fortaleza">Fortaleza </option>
   </select>
</form>

JS:

document.querySelector('select').addEventListener('change', (e) => {
   alert(`Voce selecionou a cidade ${e.target.value}`)
});
