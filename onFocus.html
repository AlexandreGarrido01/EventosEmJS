Evento onFocus

Nessa página você ira aprender um pouco sobre o evento onFocus

Como funciona o evento onFocus ?

O evento onFocus é acionado quando um elemento HTML recebe o foco, ou seja, quando o usuário seleciona ou ativa o elemento para interagir com ele. Esse evento é útil para realizar ações específicas quando um elemento recebe o foco, como alterar a cor de fundo ou exibir informações adicionais.

Abaixo temos um exemplo do evento OnFocus onde quando o ususario selecionar um campo sera emitida uma mensagem abaixo:

Exemplo

HTML:

<input type="text" placeholder="Digite seu email" autocomplete="none"/>

<p id="demo"></p>

JS:

document.querySelector('input').addEventListener('focus', (e) => {
   document.querySelector('#demo').textContent = "O input recebeu o foco";
});
document.querySelector('input').addEventListener('blur', (e) => {
   document.querySelector('#demo').textContent = "O input perdeu o foco";
});
