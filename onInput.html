Evento onInput

Nessa página você ira aprender um pouco sobre o evento onInput

Como funciona o evento onInput ?

O evento onInput é disparado sempre que o valor de um elemento HTML é alterado pelo usuário, seja por meio de digitação ou seleção de opções. Esse evento é útil para capturar as alterações feitas pelos usuários em campos de formulários, por exemplo.

Abaixo temos um exemplo do evento Input onde quando o usuario digita um nome no campo e sistema mostra qual foi esse nome logo a baixo:

Exemplo

HTML:

<form>
  <input type="text" placeholder="Digite seu nome">
</form>

<p id="demo">O nome digitado foi:</p>

JS:

document.querySelector('input').addEventListener('input', (e) => {
    document.querySelector('#demo').textContent = `O nome digitado foi: ${e.target.value}`;
});
