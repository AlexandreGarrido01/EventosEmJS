Evento onClick

Nessa página você ira aprender um pouco sobre o evento onClick

Como funciona o evento onClick?

O evento onclick é disparado quando o usuário clica em um elemento HTML. É uma forma comum de criar interações com botões, links e outros elementos clicáveis. Qualquer elemento HTML pode ser clicado para disparar o evento onclick. É possível acessar o elemento que disparou o evento usando a propriedade "this" dentro da função de resposta.

Veja a baixo um exemplo de uso do evento OnClick sendo usado para quando o botão for pressionado mandar um alerta:

Exemplo


HTML:

<button id="myButton">Clique aqui</button>

JS:

document.querySelector('#myButton').addEventListener('click', () => {
   alert('O botão foi clicado!')
});
