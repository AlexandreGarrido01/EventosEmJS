Evento onKeyUp

Nessa página você ira aprender um pouco sobre o evento onKeyUp

Como funciona o evento onKeyUp ?

O evento onKeyUp é disparado quando uma tecla é solta após ter sido pressionada. Esse evento é comumente usado para implementar interações baseadas em teclado, como ativar uma ação quando o usuário pressiona uma determinada tecla. O evento é disparado no elemento que está com o foco na página, ou seja, aquele que está pronto para receber input do usuário.

Abaixo temos um exemplo do evento OnKeyUP a onde quando o usuario soltar uma tecla o sistema irá mostrar um alerta:

Exemplo

HTML:

<input type="text"/>

JS:

document.querySelector('input').addEventListener('keyup', (e) => {
   alert(`A tecla ${e.key} foi solta`);
});
