Evento onKeyDown

Nessa página você ira aprender um pouco sobre o evento onKeyDown

Como funciona o evento onKeyDown ?

O evento onKeyDown é acionado quando uma tecla é pressionada no teclado enquanto o foco está em um elemento HTML. Esse evento é útil para capturar entradas de teclado do usuário e executar ações com base nessas entradas. É semelhante ao evento onKeyPress, mas com a diferença de que também é acionado quando uma tecla não é suportada pela entrada de caracteres.

Abaixo temos um exemplo do evento OnKeyDown onde quando o usuario aperta uma tecla dentro do campo a baixo o sistema da um alerta:

Exemplo

HTML:

<input type="text"/>

JS:

document.querySelector('input').addEventListener('keydown', (e) => {
   alert(`A tecla ${e.key} foi pressionada`);
});
